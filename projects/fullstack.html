A full-stack App collecting pizza restaurants' data

This App has react CRUD methods working with express server.

About this project:

This pizza restaurants' app is benifit for saving some interested data. You can manage these data through create, edit, search and delete methods after you login, otherwise you need to register a new account.

In the techical field, I generated these json data using JSON Generator online and save it to MongoDB. Server side includes CRUD API implementation and jwt JSON token verification. Each API point url needs to be authenticated except getting all url. The authentication part is jwt token verified which is sent with request header. Front-end interface is built by reactJS. I seperated each part such as navBar, signin, register, main as a component. I also used react-router to redirect or link to another url. Each component can get the API data through different API url. Each form also has specific validation.
