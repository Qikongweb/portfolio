A note Desktop Application

This App was built with electron framework and knexJS query builder.

About this project:

This desktop app is used to save text notes including title, tags and contents. These notes can be searched by tags or texts. It has trash which is keep the notes deleted and you can restore it back.

In the techical field, This app is built with electron framework and knexJS query builder. Sqlite3 is used to be the database.
