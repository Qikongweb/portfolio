Courses Map App

This application was an ASP.net Core 3.1 Razor Pages Project. Database was managed by Microsoft SQL server.

About this project:

This application was about display all courses' data, diploma' data, academic years's data and instrutors' data. The database was generated and managed through Microsoft SQL server which was pulled from docker hub. Scaffolded the Create, Delete, Details, and Edit pages using ASP.NET core. I created fluent APIs for each model and deployed APIs and app to Azure portal. Finally, I also buit uilt the client Ionic React app for IOS and desktop using XCode for this application.
