Online bank App

DV online bank is a functional product based on React and Java language poject. The app is designed by React Material Framework in front-end. Java Spring boot and MongoDB are used to manage data and create API end points.

About this project:

This project was to design and develop an online back application which cooprated with CGI company. I was one front-end developer in 4 persons team. The app was used for loging in a customer's account and displaying the money of serval accounts, accounts' details. The customer can transfer money from any two accounts and pay credit payment at the same time. All data were displayed by API endpoints. I was mainly responsible for designing and developing login page, account summary, details and credit payment pages using Material React. I also display all accounts details and transaction information, credit payment funtion using API endpoints.
