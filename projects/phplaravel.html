Social Feeds Website

This MVC website is built by PHP Laravel.

About this project:

This website is built by PHP laravel. The functionality is about view tour pictures and make a comment under a post. When you register a new user, it automatically send an email to a user.

I give three different user roles to administration which are user manager, theme manager and post manager. Each user can do different tasks. The website has all bootstrap watch themes which is managed by theme manager. Authenticated users can post a picuture and comment a post. Guest only can visit this website.
